1.Вам нужно организовать хранилище файлов по учебе. Создайте папки соответсвтующие вашим предметам в школе и в каждой папке создайте текстовый документ  с вашими оценками. Пользоваться можно только терминалом и базовыми командами.

Создание файлов и открытие в редакторе терминала:
touch filename.txt
nano filename.txt


2. Создайте скрипт, который будет выводить строку "Hello World! My name is User!", где вместо "User" будет подставляться твоё имя. Далее организуйте рабочую папку аналогично предыдущему заданию. После создания каждой папки выводите соответсвующее сообщение. В главной папке создайте текстовый документ с вашим именем и датой создания папки

Предпочтительно использовать for:
for VARIABLE in 1 2 3 4 5 .. N
do
    command1
    command2
    commandN
done

Как перенаправить вывод команды или данных в конец файла?
echo 'text here' >> filename
command-name >> filename
